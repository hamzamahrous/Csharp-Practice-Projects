# Shopping System (C# Console Application)

This is a simple console-based **Shopping System** built in C#.  
The main purpose of this project is to practice **Generics** and **Collections**, including:

- `List<T>`
- `Dictionary<TKey, TValue>`
- `Stack<T>`

---

## Features

### 1. View Cart

Displays all items currently added to the cart and calculates the total price.

### 2. Add Item to Cart

Shows all available products and allows the user to add a selected item to the cart.

### 3. Remove Item from Cart

Allows the user to remove an item from the cart.

### 4. Checkout

Displays the cart summary and clears the cart.

### 5. Undo Last Action

Implemented using a **Stack of tuples**.
